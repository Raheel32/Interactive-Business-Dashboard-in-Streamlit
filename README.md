# Global Superstore тАФ Interactive Business Dashboard

Task of the Data Analytics / BI mini-project series: an interactive Streamlit
dashboard for analyzing **sales**, **profit**, and **segment-wise performance** on
The Global Superstore dataset.

## ЁЯОп Task Objective

Clean and prepare the Global Superstore dataset, explore it in a Jupyter notebook,
and build an interactive Streamlit dashboard that lets a business user filter by
**Region**, **Category**, and **Sub-Category** and instantly see key KPIs:

- Total Sales
- Total Profit
- Top 5 Customers by Sales

## ЁЯзн Approach

1. **Data loading & cleaning** (notebook, Section 3):
   - Removed exact duplicate rows.
   - Filled missing `Customer Name` values.
   - Corrected invalid negative `Sales` values (sales should be a positive amount).
   - Imputed a small number of missing `Sales` values using the sub-category median.
   - Standardized text columns and derived `Profit Margin`, `Order Month`, `Order Year`.
2. **Exploratory Data Analysis** (notebook, Section 4): sales/profit by region,
   category and sub-category, monthly trend, top customers, discount vs. margin
   relationship, and segment performance тАФ each backed by a chart.
3. **Lightweight model** (notebook, Section 5): a simple linear regression
   quantifying how discount level relates to profit margin, supporting the
   discounting insight with a numeric estimate.
4. **Dashboard** (`app.py`): rebuilds the same cuts interactively тАФ
   Region / Category / Sub-Category / date-range filters in the sidebar, live
   KPI cards, and six charts (monthly trend, sales-by-region, profit by
   sub-category, top-5 customers, category comparison, discount vs. margin) plus
   a filtered-data table with CSV export.

## ЁЯУИ Results & Findings

- **Technology** and **Office Supplies** are the strongest profit contributors;
  several **Furniture** sub-categories (e.g., Bookcases, Tables) run at a net
  loss, driven mainly by heavy discounting on already low-margin items.
- **Discounting has a clear negative relationship with profit margin** тАФ the
  fitted regression shows profit margin dropping several points for every 10%
  of additional discount, suggesting a discount cap could protect profitability.
- **Sales are concentrated among a small set of high-value customers**, an
  80/20-style pattern that supports targeted retention efforts for top accounts.
- **Regional sales are fairly balanced**, but profit contribution differs more
  across regions тАФ a signal for region-specific pricing/discount policy.
- **Consumer** segment drives the most volume; **Corporate**/**Home Office**
  orders trend toward higher average order value.

Full analysis, charts, and code walkthrough: `notebook/Global_Superstore_Analysis.ipynb`.
